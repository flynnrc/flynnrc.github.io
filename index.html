Twilio Receiver Service

I created a service that listens for twilio text messages and deployed it to the Microsoft Azure Cloud. The service works when linked to a twilio account and it bounces SSM or MSS messages sent from a twilio number to a listening program. I found it useful for automating an application that required a local phone number for authentication. There is a connection-hub for minor parallelization. (Repo versions: (1) C# linqpad (2) C# Visual Studio Project).
